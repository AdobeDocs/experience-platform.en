# Manage sources dataflows in the UI

You can use the *Sources workspace* in the Adobe Experience Platform user interface to manage your existing sources dataflows. Use the [!UICONTROL Dataflows] page to access a centralized view of your organization's existing dataflows and search, filter, organize, and take actions on individual flows.

## Get started

Before you begin, ensure that you have the following:

- Access to Adobe Experience Platform.
- Both **[!UICONTROL View Sources]** and **[!UICONTROL Manage Sources]** permissions.

It is helpful to have an understanding of the following Experience Platform features and concepts before working with the object navigation tools in the sources workspace:

- [Sources](../../home.md): Learn how to connect, manage, and monitor external data sources in Experience Platform.
- [Sandboxes](../../../sandboxes/home.md): Discover how sandboxes let you develop and test different projects in isolated environments.
- [Administrative tags](../../../administrative-tags/overview.md):

## Access your source dataflows

In the Experience Platform UI, select **[!UICONTROL Sources]** from the left navigation to access the Sources workspace. Next, **[!UICONTROL Dataflows]** from the top header. The *[!UICONTROL Dataflows]* page displays a list of existing dataflows in your organization.

## Organize dataflows with tags

You can use tags to organize your dataflows and browse through them more efficiently. Tags can also be used as a filter to narrow the dataflow list.

1. Locate the dataflow you want to organize.
2. Open the row action menu or tag management entry point.
3. Add, remove, or update tags.
4. Save your changes.
5. Use the Tags filter to find similarly tagged dataflows.

## Resize table columns

You can resize table columns in the Dataflows page to display more of a dataflow's metadata, such as names, account details, or target dataset information.

1. Hover over the edge of a column header
2. Drag the column boundary to increase or decrease the column width.

## Take action on a dataflow

After locating the right dataflow, use the row action menu or select the dataflow directly to continue with related tasks such as:

- Updating a dataflow
- Editing a schedule
- Disabling a dataflow
- Deleting a dataflow
- Monitoring a dataflow
- Resuming a draft, if applicable
