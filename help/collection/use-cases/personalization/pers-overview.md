# Personalization use cases overview

The Adobe Experience Platform Web SDK enables businesses to deliver a wide variety of personalization use cases to their web properties. You can seamlessly connect data collection, decisioning, and activation processes across Adobe solutions like Target, Journey Optimizer, and Audience Manager.

The SDK supports flexible architectures, including client-side, server-side, and hybrid models, making it possible to fetch and render personalized offers that meet most organization's use cases.

## Web SDK samples

In addition to the doc pages in this folder, Adobe maintains a repository of sample applications that you can reference. See [Web SDK samples](https://github.com/adobe/alloy-samples/) on GitHub for more personalization use cases, including:

* Client-side personalization
* Server-side personalization
* Hybrid personalization
* Personalization using single-page applications
