# Brightscript overview

The **Adobe Experience Platform Roku SDK** allows you to send data to a [datastream](/help/datastreams/overview.md) from a Roku device using Brightscript.

The SDK and documentation for this library are hosted on GitHub:

**[Adobe Experience Platform Roku SDK](https://github.com/adobe/aepsdk-roku)**
